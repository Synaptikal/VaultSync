# VaultSync User Manual

## Getting Started

1.  **Launch the Server**:
    Run `vaultsync` executable. It listens on port 3000 by default.

2.  **Launch the App**:
    Open the VaultSync mobile/desktop application.

3.  **Login**:
    Enter your username and password. Default admin credentials should be set up during installation.

## POS Terminal

- **Add to Cart**: Tap on a product in the grid to add it to the cart.
- **Adjust Quantity**: Use the +/- buttons in the cart list.
- **Checkout**: Click "Checkout" to finalize the sale.

## Inventory Management

- **View Inventory**: Navigate to the Inventory tab to see current stock levels.
- **Add Stock**: (Coming Soon) Scan a barcode or search the catalog to add items.

## Customer Management

- **Customer List**: View all registered customers and their store credit.
- **New Customer**: (Coming Soon) Register a new customer.

## Troubleshooting

- **Connection Error**: Ensure the backend server is running and the device is on the same network.
- **Login Failed**: Check your credentials. If you forgot your password, contact the system administrator.
