# VaultSync Mobile Frontend

This is the Flutter frontend for VaultSync.

## Prerequisites

- Flutter SDK (latest stable)
- Android Studio / Xcode (for emulators)

## Setup

1.  Navigate to the `frontend` directory:
    ```bash
    cd frontend
    ```
2.  Install dependencies:
    ```bash
    flutter pub get
    ```
3.  Run the application:
    ```bash
    flutter run
    ```

## Structure

- `lib/src/features`: Contains feature-specific code (Authentication, Dashboard, etc.)
- `lib/src/app.dart`: Main app configuration and routing.
- `lib/main.dart`: Entry point.
